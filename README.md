
# Simple Auth App
Mini project for crud employee and approval reimbursement based on role
## Run Locally

Clone the project

```bash
  git clone https://github.com/azkiya/simple-auth-app
```

Go to the project directory

```bash
  cd simple-auth-app
```

Install dependencies

```bash
  composer install
```
```bash
  npm install
```

Setting environment

```bash
  create file .env minimal require from env.example & create DB 
```

Migration

```bash
  php artisan migrate
```

Run seeder

```bash
  php artisan db:seed
```

Run Generate Key

```bash
  php artisan key:generate
```

Run Storage Link

```bash
  php artisan storage:link
```

Start the server

```bash
  php artisan serve
```

Build FE

```bash
  npm run dev
```


## Tech Stack

**Server:** PHP

**Framework** Laravel 10
